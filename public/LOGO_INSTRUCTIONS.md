# LucidLife 로고 이미지 추가 방법

## 중요: 이미지 파일을 추가해야 합니다!

현재 Header 컴포넌트는 `/lucidlife-logo.png` 경로의 이미지를 참조하고 있습니다.

## 이미지 파일 추가 단계:

1. **첨부하신 LucidLife 로고 이미지를 다운로드하세요**
   - 금색 로고와 "LUCIDLIFE" 텍스트가 포함된 이미지

2. **이미지 파일을 `public` 폴더에 저장하세요**
   - 파일 경로: `public/lucidlife-logo.png`
   - 파일명: `lucidlife-logo.png` (정확히 이 이름으로)

3. **이미지 형식:**
   - PNG (권장) - 투명 배경 가능
   - SVG (가능) - 벡터 형식, 확대해도 선명함
   - JPG (가능) - 배경이 필요한 경우

4. **이미지 크기 권장사항:**
   - 최소: 200x200px
   - 권장: 400x400px 이상 (고해상도)
   - 비율: 1:1 (정사각형) 또는 가로가 약간 긴 형태

## SVG를 사용하는 경우:

SVG 파일을 사용하려면:
1. 파일명을 `lucidlife-logo.svg`로 저장
2. `src/components/Header.tsx` 파일에서 경로를 `/lucidlife-logo.svg`로 변경

## 현재 로고 크기 설정:

- 모바일: 80px (h-20)
- 태블릿: 96px (h-24)
- 데스크톱: 112px (h-28)
- 큰 화면: 128px (h-32)

이미지 파일을 추가하면 홈페이지 좌측 상단에 로고가 표시됩니다!
